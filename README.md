## 📖 Overview

**Mnemonic Seed Generator** is a lightweight utility for the local generation of cryptographically secure seed phrases. The program strictly follows the **BIP39** standard and utilizes a system Cryptographically Secure Pseudo-Random Number Generator (CSPRNG) to ensure maximum entropy.

## ✨ Core Features

*   🛡️ **Cryptographic Security:** Utilizes CSPRNG for entropy generation.
*   📏 **BIP39 Standard Support:** Generates phrases of 12, 15, 18, 21, and 24 words.
*   🔌 **Offline Generation:** The program requires no internet connection and does not transmit data to any external servers.
*   🌐 **Multi-language:** Supports the standard English BIP39 wordlist, as well as additional language packs.
*   🧹 **Memory Wiping:** Automatically clears the generated phrase from RAM upon closing the utility or terminating the session.
